Aperçu du projet

EventHub est une application web moderne développée avec React pour le front-end et Laravel pour le back-end. Elle permet la gestion, la planification et la consultation d'événements dans un cadre professionnel ou communautaire.

L'application offre une interface intuitive pour créer, modifier et consulter des événements, gérer les inscriptions des participants et les informations logistiques (lieu, date, intervenants…). Elle permet également un accès en temps réel aux informations sur les événements via une API Laravel et fournit un tableau de bord interactif pour les utilisateurs et les organisateurs.

Défis et solutions

Défis rencontrés

  • Gestion des inscriptions en temps réel

    Difficulté à synchroniser les inscriptions et à gérer les places disponibles en temps réel.

  • Authentification sécurisée

    Mise en place d'un système d'authentification robuste avec gestion des rôles et permissions.

  • Génération de QR codes

    Intégration d'un système de génération et validation de QR codes pour les billets d'événements.

Solutions implémentées

  • WebSockets pour les mises à jour en temps réel

    Utilisation de Laravel Echo et Pusher pour synchroniser les inscriptions et mettre à jour les places disponibles instantanément.

  • JWT et Sanctum pour l'authentification

    Implémentation de JWT pour l'API et Laravel Sanctum pour la gestion des sessions et des tokens d'API.

  • Bibliothèque QR Code

    Intégration de la bibliothèque QR Code JS côté client et Simple QR Code côté serveur pour générer et valider les codes.

EventController.php (Laravel)


public function store(EventRequest $request)
{
    $validated = $request->validated();
    
    $event = new Event();
    $event->title = $validated['title'];
    $event->description = $validated['description'];
    $event->start_date = $validated['start_date'];
    $event->end_date = $validated['end_date'];
    $event->location = $validated['location'];
    $event->capacity = $validated['capacity'];
    $event->user_id = auth()->id();
    $event->save();
    
    // Generate QR code for the event
    $qrCode = QrCode::size(300)
        ->generate(route('events.show', $event->id));
    
    // Trigger real-time notification
    event(new EventCreated($event));
    
    return response()->json([
        'message' => 'Event created successfully',
        'event' => new EventResource($event),
        'qr_code' => base64_encode($qrCode)
    ], 201);
}
