Site éducatif full-stack

Plateforme complète développée avec J2EE, HTML, CSS et JavaScript

Aperçu du projet

Un site web éducatif interactif développé en full-stack utilisant J2EE pour le back-end et les technologies modernes (HTML, CSS, JavaScript, Sass) pour le front-end. Cette plateforme offre une expérience d'apprentissage complète avec une interface intuitive et des fonctionnalités serveur robustes.
