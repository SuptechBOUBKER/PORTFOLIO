NextStepJobs

Une plateforme de recrutement en ligne développée avec Django et SQLite Studio

Aperçu du projet

NextStepJobs est une plateforme de recrutement en ligne qui connecte les chercheurs d'emploi aux employeurs. Développée avec Django pour le back-end et SQLite pour la gestion des données, elle offre une interface utilisateur intuitive et des fonctionnalités robustes pour faciliter le processus de recrutement.

Technologies clés

Django SQLite HTML CSS JavaScript

Fonctionnalités

  • Recherche avancée d'offres
  • Tableaux de bord utilisateur
  • Authentification sécurisée
